CareerFlow

View Code →

CareerFlow is a full-stack job application tracking platform built with React and Firebase. The application allows users to manage and monitor their job search through a clean dashboard interface where applications can be added, edited, and tracked across hiring stages including Applied, Interview, Offer, and Rejected. Data is securely stored using Firebase Firestore, allowing users to maintain persistent job tracking sessions.
